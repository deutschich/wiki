# Über CheapDesign
CheapDesign (ehemaliger Name *CheapMC*) ist ein **Minecraft Resource Pack**, das Texturen zu **Günstig aussehenden Texturen** ändert und das Spiel sehr billig aussehen lässt, ähnlich wie auf manchen *sehr günstigen Online-Shops*. Das Projekt wird von deutschich (aka. User404_, MineMeeer, MineMeer) gepflegt. Jeder kann zum Projekt beitragen (Mehr Details unter [Contributing](contributing.md))

# Wie kann ich dieses Pack nutzen?

Um CheapDesign zu nutzen, hängt es davon ab, welche Plattform Sie verwenden.

## Unter Windows

Wenn Sie unter Windows sind, können Sie dieses Pack in **Minecraft: Java Edition** und/oder in **Minecraft: Bedrock Edition** (früher **Minecraft for Windows**) verwenden.

### Minecraft: Java Edition

Um dieses Pack in **Minecraft: Java Edition** zu nutzen, müssen Sie das Pack auf den offiziellen Download-Seiten herunterladen: [Modrinth](https://modrinth.com/project/Jde5u9cx), [CurseForge](https://www.curseforge.com/minecraft/texture-packs/cheapdesign) oder [GitHub](https://github.com/deutschich/CheapMC/releases/latest) und die ZIP-Datei in den `.minecraft\resourcepacks`-Ordner legen.

#### Wie öffne ich den `.minecraft` Ordner?

Um in den `.minecraft` Ordner zu gelangen, öffnen Sie die **Windows Suche** und geben ein
```
```batch
%appdata%\.minecraft
```
```
Jetzt befinden Sie sich im `.minecraft`-Ordner.

### Minecraft: Bedrock Edition

Um dieses Pack in **Minecraft: Bedrock Edition** zu nutzen, müssen Sie das Pack auf der offiziellen Download-Seite [CurseForge](https://www.curseforge.com/minecraft-bedrock/texture-packs/cheapdesign-bedrock) herunterladen und öffnen. Minecraft sollte das Spiel öffnen und die Datei importieren.
