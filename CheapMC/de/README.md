# Über CheapDesign
CheapDesign (Altnamen *CheapMC*) ist ein **Minecraft Resource Pack**, das Texturen zu **preiswerten Texturen** ändert, sodass das Spiel sehr billig aussieht, ähnlich wie in einigen *sehr billigen Online-Shops*. Das Projekt wird von deutschich (aka. User404_, MineMeeer, MineMeer) gepflegt. Jeder kann zum Projekt beitragen (Mehr Details unter [Contributing](contributing.md))

# Wie kann ich dieses Pack nutzen?

Um CheapDesign zu nutzen, hängt es davon ab, welche Plattform Sie verwenden.

## Auf Windows

Wenn Sie Windows nutzen, können Sie dieses Pack in **Minecraft: Java Edition** und/oder in **Minecraft: Bedrock Edition** (früher **Minecraft for Windows**) verwenden.

### Minecraft: Java Edition

Um dieses Pack in **Minecraft: Java Edition** zu nutzen, müssen Sie das Pack auf den offiziellen Downloadseiten herunterladen: [Modrinth](https://modrinth.com/project/Jde5u9cx), [CurseForge](https://www.curseforge.com/minecraft/texture-packs/cheapdesign) oder [GitHub](https://github.com/deutschich/CheapMC/releases/latest) und die ZIP-Datei in den `.minecraft\resourcepacks`-Ordner legen.

#### Wie öffne ich den Ordner `.minecraft`?

Um in den Ordner `.minecraft` zu gelangen, öffnen Sie die **Windows-Suche** und geben ein:
```
```batch
%appdata%\.minecraft
```
```
Jetzt befinden Sie sich im Ordner `.minecraft`.

### Minecraft: Bedrock Edition

Um dieses Pack in **Minecraft: Bedrock Edition** zu nutzen, müssen Sie das Pack auf der offiziellen Downloadseite [CurseForge](https://www.curseforge.com/minecraft-bedrock/texture-packs/cheapdesign-bedrock) herunterladen und öffnen. Minecraft sollte das Spiel öffnen und die Datei importieren.
