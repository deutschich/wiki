## Wie man beitragen kann

Vielen Dank für Ihr Interesse daran, zu **CheapDesign** beizutragen! Wir freuen uns über von der Community erstellte Texturen, die die billige, niedrigqualitative Ästhetik des Packs widerspiegeln.

### Was Sie beitragen können
- Neue oder verbesserte Texturen für **Minecraft Java und/oder Bedrock Edition**
- Die Texturen sollen absichtlich *billig*, simpel oder unvollkommen wirken
- Kreative Interpretationen sind willkommen, solange sie dem Gesamtstil entsprechen

### Richtlinien für Texturen
- Auflösung: **16×16** (kein Upscaling)
- Format: **PNG** mit Transparenz, wo nötig
- Texturen müssen dem **Minecraft Java Edition Resource‑Pack‑Format** entsprechen
- Legen Sie alle Dateien im korrekten Verzeichnis ab:  
  `assets/minecraft/textures/`  
  (z. B. `block/`, `item/`, `entity/`, usw.)
- Kein Inhalt aus anderen Texture‑Packs, außer er ist kompatibel mit **CC BY‑SA 4.0**
- Verwenden Sie keine KI‑generierten Texturen
- Die Texturen sollten im Spiel gut lesbar sein

### Lizenz
Dieses Projekt ist unter **CC BY‑SA 4.0** lizenziert. Durch das Beitragen erklären Sie sich damit einverstanden, dass:
- Ihre Arbeit original oder rechtlich kompatibel ist
- Ihr Beitrag unter **CC BY‑SA 4.0** veröffentlicht wird
- Andere Ihre Arbeit unter der gleichen Lizenz remixen oder redistribuieren dürfen

### So senden Sie Änderungen ein
1. Forken Sie das offizielle Repository:  
   https://github.com/deutschich/CheapMC
2. Erstellen Sie einen neuen Branch oder nutzen Sie Ihr Fork
3. Fügen Sie Ihre Texturen in die richtigen **Minecraft Java Resource‑Pack‑Ordner** ein
4. Erstellen Sie einen **Pull Request in den `dev` Branch**  
   ⚠️ Pull Requests in andere Branches werden nicht akzeptiert

### Review‑Prozess
Alle Pull Requests werden geprüft auf:
- Richtiges Ordner‑Layout
- Stil‑Konsistenz
- Technische Korrektheit
- Lizenz‑Kompatibilität

Vielen Dank, dass Sie dabei helfen, CheapDesign zu verbessern!
