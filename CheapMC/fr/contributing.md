## Comment contribuer

Merci de votre intérêt pour contribuer à **CheapDesign** !  
Nous accueillons les textures créées par la communauté qui s’inscrivent dans l’esthétique économique et de faible qualité du pack.

### Ce que vous pouvez contribuer
- De nouvelles textures ou des textures améliorées pour **Minecraft Java et/ou Bedrock Edition**  
- Les textures doivent intentionnellement paraître *économiques*, simples ou imparfaites  
- Les interprétations créatives sont les bienvenues tant qu’elles correspondent au style global  

### Directives pour les textures
- Résolution : **16×16** (pas de suréchantillonnage)  
- Format : **PNG** avec transparence si nécessaire  
- Les textures doivent respecter le format du pack de ressources **Minecraft Java Edition**  
- Placez tous les fichiers dans le répertoire correct à l’intérieur de ``assets/minecraft/textures/`` (par ex. ``block/``, ``item/``, ``entity/``, etc.)  
- Aucun contenu provenant d’autres packs de textures, sauf s’il est compatible avec **CC BY-SA 4.0**  
- N’utilisez pas de textures générées par IA  
- Gardez les textures lisibles dans le jeu  

### Licence
Ce projet est sous licence **CC BY-SA 4.0**.  
En contribuant, vous acceptez que :
- Votre travail est original ou légalement compatible  
- Votre contribution sera publiée sous **CC BY-SA 4.0**  
- D’autres peuvent remixer ou redistribuer votre travail sous la même licence  

### Comment soumettre des modifications
1. Forkez le dépôt officiel : https://github.com/deutschich/CheapMC  
2. Créez une nouvelle branche ou utilisez votre fork  
3. Ajoutez vos textures aux dossiers de pack de ressources **Minecraft Java** appropriés  
4. Créez une **Pull Request vers la branche ``dev``** ⚠️ Les Pull Requests vers d’autres branches ne seront pas acceptées  

### Processus de revue
Toutes les Pull Requests seront examinées pour :
- La structure correcte des dossiers  
- La cohérence du style  
- La conformité technique  
- La compatibilité de la licence  

Merci de contribuer à l’amélioration de CheapDesign !
