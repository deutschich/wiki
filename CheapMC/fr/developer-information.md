## Questions à propos de ce pack

Les questions concernant ce pack peuvent être posées via [GitHub Issues](https://github.com/deutschich/CheapMC/issues) ou sur mon **pas encore disponible** *serveur Discord*.

## Comment fonctionne ce pack pour Java & Bedrock

Je crée ce pack principalement pour Java, mais j’ai trouvé un convertisseur Java‑to‑Bedrock et je l’utilise. Si vous souhaitez ajouter des textures, il suffit de les ajouter au pack Java via une Pull Request. Plus d’informations à ce sujet sont disponibles sur [Contributing](contributing).
