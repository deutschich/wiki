# À propos de CheapDesign

CheapDesign (ancien nom *CheapMC*) est un **pack de ressources Minecraft** qui remplace les textures par des **textures à petit prix**, donnant au jeu un aspect très bon marché, comme sur certaines *boutiques en ligne très bon marché*. Le projet est maintenu par deutschich (alias User404_, MineMeeer, MineMeer). Tout le monde peut contribuer au projet (plus de détails sur [Contributing](contributing.md)).

# Comment puis-je utiliser ce pack ?

Pour utiliser CheapDesign, cela dépend de la plateforme que vous utilisez.

## Sous Windows

Si vous êtes sous Windows, vous pouvez utiliser ce pack sur **Minecraft : Edition Java** et/ou **Minecraft : Edition Bedrock** (anciennement **Minecraft pour Windows**).

### Minecraft : Edition Java

Pour utiliser ce pack sur **Minecraft : Edition Java**, vous devez télécharger le pack sur les sites de téléchargement officiels : [Modrinth](https://modrinth.com/project/Jde5u9cx), [CurseForge](https://www.curseforge.com/minecraft/texture-packs/cheapdesign) ou [GitHub](https://github.com/deutschich/CheapMC/releases/latest) et placer le fichier ZIP dans le dossier `.minecraft\resourcepacks`.

#### Comment ouvrir le dossier `.minecraft` ?

Pour accéder au dossier `.minecraft`, vous devez ouvrir la **Recherche Windows** et taper

```batch
%appdata%\.minecraft
```
. Vous êtes maintenant dans le dossier `.minecraft`.

### Minecraft : Edition Bedrock

Pour utiliser ce pack sur **Minecraft : Edition Java**, vous devez télécharger le pack sur le site de téléchargement officiel [CurseForge](https://www.curseforge.com/minecraft-bedrock/texture-packs/cheapdesign-bedrock) et l'ouvrir. Minecraft devrait s'ouvrir et importer le fichier.
