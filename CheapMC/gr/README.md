# Σχετικά με το CheapDesign
Το CheapDesign (Παλαιό Όνομα *CheapMC*) είναι ένα **Minecraft Resource Pack** που αλλάζει τις υφές σε **Χαλακές Υφές**, που κάνουν το παιχνίδι να φαίνεται πολύ φθηνό, όπως σε κάποιες *πολύ φθηνές διαδικτυακές αγορές*. Το έργο διατηρείται από τον deutschich (aka. User404_, MineMeeer, MineMeer). Όλοι μπορούν να συνεισφέρουν στο έργο (Περισσότερες λεπτομέρειες στο [Contributing](contributing.md))

# Πώς μπορώ να χρησιμοποιήσω αυτό το Pack;

Για να χρησιμοποιήσετε το CheapDesign, εξαρτάται από τη πλατφόρμα που χρησιμοποιείτε.

## Σε Windows

Αν είστε σε Windows, μπορείτε να χρησιμοποιήσετε το Pack σε **Minecraft: Java Edition** και/ή σε **Minecraft: Bedrock Edition** (πρώην **Minecraft for Windows**).

### Minecraft: Java Edition

Για να χρησιμοποιήσετε το Pack σε **Minecraft: Java Edition**, πρέπει να κατεβάσετε το Pack από τους επίσημους ιστότοπους λήψης: [Modrinth](https://modrinth.com/project/Jde5u9cx), [CurseForge](https://www.curseforge.com/minecraft/texture-packs/cheapdesign) ή [GitHub](https://github.com/deutschich/CheapMC/releases/latest) και να τοποθετήσετε το αρχείο ZIP στον φάκελο `.minecraft\resourcepacks`.

#### Πώς να ανοίξετε τον φάκελο `.minecraft`;

Για να μπείτε στον φάκελο `.minecraft`, πρέπει να ανοίξετε την **Windows Search** και να πληκτρολογήσετε
```batch
%appdata%\.minecraft
```
. Τώρα βρίσκεστε στον φάκελο `.minecraft`.

### Minecraft: Bedrock Edition

Για να χρησιμοποιήσετε το Pack σε **Minecraft: Bedrock Edition**, πρέπει να κατεβάσετε το Pack από τον επίσημο ιστότοπο λήψης [CurseForge](https://www.curseforge.com/minecraft-bedrock/texture-packs/cheapdesign-bedrock) και να το ανοίξετε. Το Minecraft θα πρέπει να ανοίξει και να εισάγει το αρχείο.
