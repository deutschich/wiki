# Σχετικά με το CheapDesign
Το CheapDesign (Παλιό Όνομα *CheapMC*) είναι ένα **Minecraft Resource Pack** που αλλάζει τα textures σε **οικονομικά textures**, κάνοντας το παιχνίδι να φαίνεται πολύ φθηνό, όπως σε μερικά *πολύ φθηνά online καταστήματα*.  
Το έργο συντηρείται από τον deutschich (aka. User404_, MineMeeer, MineMeer). Οποιοσδήποτε μπορεί να συνεισφέρει στο έργο (Περισσότερες λεπτομέρειες στο [Contributing](contributing.md)).

# Πώς μπορώ να χρησιμοποιήσω αυτό το Pack;

Για να χρησιμοποιήσετε το CheapDesign, εξαρτάται από το ποια πλατφόρμα χρησιμοποιείτε.

## Σε Windows

Αν είστε σε Windows, μπορείτε να χρησιμοποιήσετε αυτό το Pack στο **Minecraft: Java Edition** και/ή στο **Minecraft: Bedrock Edition** (πρώην **Minecraft for Windows**).

### Minecraft: Java Edition

Για να χρησιμοποιήσετε αυτό το Pack στο **Minecraft: Java Edition**, πρέπει να κατεβάσετε το Pack από τους επίσημους ιστότοπους λήψης: [Modrinth](https://modrinth.com/project/Jde5u9cx), [CurseForge](https://www.curseforge.com/minecraft/texture-packs/cheapdesign) ή [GitHub](https://github.com/deutschich/CheapMC/releases/latest) και να τοποθετήσετε το αρχείο ZIP στον φάκελο `.minecraft\resourcepacks`.

#### Πώς να ανοίξετε το φάκελο `.minecraft`;

Για να μπείτε στον φάκελο `.minecraft`, πρέπει να ανοίξετε την **Αναζήτηση Windows** και να πληκτρολογήσετε
```
```batch
%appdata%\.minecraft
```
```
Τώρα βρίσκεστε στον φάκελο `.minecraft`.

### Minecraft: Bedrock Edition

Για να χρησιμοποιήσετε αυτό το Pack στο **Minecraft: Java Edition**, πρέπει να κατεβάσετε το Pack από τον επίσημο ιστότοπο λήψης [CurseForge](https://www.curseforge.com/minecraft-bedrock/texture-packs/cheapdesign-bedrock) και να το ανοίξετε. Το Minecraft θα πρέπει να ανοίξει και να εισάγει το αρχείο.
