## Πώς να συνεισφέρετε

Ευχαριστούμε για το ενδιαφέρον σας να συνεισφέρετε στο **CheapDesign**!  
Καλωσορίζουμε υφές που δημιουργήθηκαν από την κοινότητα και ταιριάζουν με την φθηνή, χαμηλής ποιότητας αισθητική του πακέτου.

### Τι μπορείτε να συνεισφέρετε
- Νέες ή βελτιωμένες υφές για το **Minecraft Java και/ή Bedrock Edition**  
- Οι υφές θα πρέπει να φαίνονται *φθηνές*, απλές ή ελλιπείς  
- Δημιουργικές ερμηνείες είναι ευπρόσδεκτες εφόσον ταιριάζουν με το συνολικό στυλ

### Οδηγίες υφής
- Επίλυση: **16×16** (χωρίς αναβάθμιση)  
- Μορφή: **PNG** με διαφάνεια όπου χρειάζεται  
- Οι υφές πρέπει να ακολουθούν τη μορφή **Minecraft Java Edition resource pack**  
- Τοποθετήστε όλα τα αρχεία στον σωστό κατάλογο μέσα στο ``assets/minecraft/textures/`` (π.χ. ``block/``, ``item/``, ``entity/``, κ.λπ.)  
- Κανένας περιεχόμενος από άλλα πακέτα υφών, εκτός εάν είναι συμβατός με το **CC BY-SA 4.0**  
- Μην χρησιμοποιείτε υφές που δημιουργήθηκαν με AI  
- Κρατήστε τις υφές ευανάγνωστες στο παιχνίδι

### Άδεια
Αυτό το έργο είναι αδειοδοτημένο υπό την **CC BY-SA 4.0**.  
Με τη συνεισφορά, συμφωνείτε ότι:
- Η εργασία σας είναι πρωτότυπη ή νόμιμα συμβατή  
- Η συνεισφορά σας θα δημοσιευτεί υπό την **CC BY-SA 4.0**  
- Άλλοι μπορούν να αναμιγνύουν ή να διανεμηθούν την εργασία σας υπό την ίδια άδεια

### Πώς να υποβάλετε αλλαγές
1. Κάντε fork του επίσημου αποθετηρίου: <https://github.com/deutschich/CheapMC>  
2. Δημιουργήστε ένα νέο κλάδο ή χρησιμοποιήστε το fork σας  
3. Προσθέστε τις υφές σας στους σωστούς φάκελους **Minecraft Java resource pack**  
4. Δημιουργήστε ένα **Pull Request στην κλάδο ``dev``** ⚠️ Τα Pull Requests σε άλλες κλάδους δεν θα γίνει αποδεκτά

### Διαδικασία αξιολόγησης
Όλα τα Pull Requests θα αξιολογηθούν για:
- Έξυπνη δομή φακέλων  
- Συνεπές στυλ  
- Τεχνική ορθότητα  
- Συμβατότητα άδειας

Ευχαριστούμε που βοηθάτε στη βελτίωση του CheapDesign!
