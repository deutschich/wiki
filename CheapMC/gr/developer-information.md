# Πληροφορίες Προγραμματιστών
Ακολουθούν κάποιες πληροφορίες για τους *προγραμματιστές* αυτού του πακέτου πόρων.

## Ποιοι είναι οι κύριοι προγραμματιστές;

Αυτήν τη στιγμή υπάρχει μόνο ένας κύριος προγραμματιστής στο Plugin, και αυτός είμαι εγώ deutschich (ακόμα και ως User404_, MineMeeer, MineMeer).

## Ερωτήσεις για αυτό το Πακέτο

Οι ερωτήσεις για αυτό το Πακέτο μπορούν να γίνουν μέσω [GitHub Issues](https://github.com/deutschich/CheapMC/issues) ή στο **δεν διαθέσιμο ακόμη** *Discord Server* μου.

## Πώς λειτουργεί αυτό το Πακέτο για Java & Bedrock

Δημιουργώ αυτό το Πακέτο κυρίως για Java, αλλά βρήκα έναν μετατροπέα Java 2 Bedrock Pack και τον χρησιμοποιώ. Αν θέλετε να προσθέσετε υφές, πρέπει απλώς να τις προσθέσετε στο Java Pack με ένα Pull Request. Περισσότερες πληροφορίες για αυτό μπορείτε να βρείτε στο [Contributing](contributing).
