This is the official Wiki for CheapDesign! Here you can get Information to the Resource Pack, to the Updates and what we are currently working on.

# About CheapDesign
CheapDesign (Old Name *CheapMC*) is a **Minecraft Resource Pack** which changes textures to **Cheap Looking Textures**, which makes the game looking very cheap, like on some *very cheap online shops*. The Project is maintained by deutschich (aka. User404_, MineMeeer, MineMeer). Everyone can contribute to the project (More details at [Contributing](contributing.md))

# How can I use this Pack?

To use CheapDesign, it depends on which Platform you are. 

## On Windows

If you are on Windows, you can use this Pack on **Minecraft: Java Edition** and/or on **Minecraft: Bedrock Edition** (formerly **Minecraft for Windows**).

### Minecraft: Java Edition

To use this Pack on **Minecraft: Java Edition**, you have to Download the Pack on the Official Download Sites: [Modrinth](https://modrinth.com/project/Jde5u9cx), [CurseForge](https://www.curseforge.com/minecraft/texture-packs/cheapdesign) or [GitHub](https://github.com/deutschich/CheapMC/releases/latest) and place the ZIP-File in the `.minecraft\resourcepacks`-Folder.

#### How to open the `.minecraft` folder?

To get into the `.minecraft` folder, you have to open the **Windows Search** and type in
```batch
%appdata%\.minecraft
```
. Now you are in the `.minecraft`-Folder.

# Can I create own variations of this Pack?

Yes, you can create your own version if you add a clearly visible link to our repository as attribution.
