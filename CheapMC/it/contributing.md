## Come contribuire

Grazie per l'interesse a contribuire a **CheapDesign**!  
Accogliamo texture create dalla comunità che si adattino all'estetica economica e di bassa qualità del pacchetto.

### Cosa puoi contribuire
- Nuove o migliorate texture per **Minecraft Java e/o Bedrock Edition**
- Le texture dovrebbero apparenze volutamente *economiche*, semplici o imperfette
- Le interpretazioni creative sono benvenute purché corrispondano allo stile complessivo

### Linee guida per le texture
- Risoluzione: **16×16** (nessun ridimensionamento)
- Formato: **PNG** con trasparenza dove necessario
- Le texture devono seguire il formato del resource pack **Minecraft Java Edition**
- Posiziona tutti i file nella directory corretta all'interno di **`assets/minecraft/textures/`** (es. **`block/`**, **`item/`**, **`entity/`**, ecc.)
- Nessun contenuto da altri pacchetti di texture a meno che non sia compatibile con **CC BY-SA 4.0**
- Non usare texture generate da IA
- Mantieni le texture leggibili in gioco

### Licenza
Questo progetto è concesso in licenza sotto **CC BY-SA 4.0**.  
Contribuendo, accetti che:
- Il tuo lavoro è originale o legalmente compatibile
- Il tuo contributo sarà pubblicato sotto **CC BY-SA 4.0**
- Altri possono remixare o ridistribuire il tuo lavoro sotto la stessa licenza

### Come inviare modifiche
1. Forka il repository ufficiale: https://github.com/deutschich/CheapMC  
2. Crea un nuovo branch o utilizza il tuo fork  
3. Aggiungi le tue texture alle cartelle corrette del resource pack **Minecraft Java**  
4. Crea una **Pull Request nel branch `dev`** ⚠️ Le Pull Request verso altri branch non verranno accettate

### Processo di revisione
Tutte le Pull Request saranno revisionate per:
- Struttura corretta delle cartelle
- Coerenza dello stile
- Correttezza tecnica
- Compatibilità con la licenza

Grazie per aver aiutato a migliorare CheapDesign!
