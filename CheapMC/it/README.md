# Informazioni su CheapDesign
CheapDesign (ex *CheapMC*) è un **Minecraft Resource Pack** che cambia le texture in **texture dall’aspetto economico**, facendo apparire il gioco molto economico, come in alcuni *negozi online estremamente economici*. Il progetto è mantenuto da deutschich (aka. User404_, MineMeeer, MineMeer). Chiunque può contribuire al progetto (maggiori dettagli su [Contributing](contributing.md))

# Come posso usare questo Pack?

Per usare CheapDesign, dipende dalla piattaforma su cui ti trovi.

## Su Windows

Se sei su Windows, puoi usare questo Pack su **Minecraft: Java Edition** e/o su **Minecraft: Bedrock Edition** (precedentemente **Minecraft for Windows**).

### Minecraft: Java Edition

Per usare questo Pack su **Minecraft: Java Edition**, devi scaricare il Pack sui siti di download ufficiali: [Modrinth](https://modrinth.com/project/Jde5u9cx), [CurseForge](https://www.curseforge.com/minecraft/texture-packs/cheapdesign) o [GitHub](https://github.com/deutschich/CheapMC/releases/latest) e posizionare il file ZIP nella cartella `.minecraft\resourcepacks`.

#### Come aprire la cartella `.minecraft`?

Per entrare nella cartella `.minecraft`, devi aprire la **Cerca di Windows** e digitare
```batch
%appdata%\.minecraft
```
. Ora sei nella cartella `.minecraft`.

### Minecraft: Bedrock Edition

Per usare questo Pack su **Minecraft: Bedrock Edition**, devi scaricare il Pack sul sito di download ufficiale [CurseForge](https://www.curseforge.com/minecraft-bedrock/texture-packs/cheapdesign-bedrock) e aprirlo. Minecraft dovrebbe aprirsi e importare il file.

# Posso creare variazioni proprie di questo Pack?

Sì, puoi creare la tua versione se aggiungi un link chiaramente visibile al nostro repository come attribuzione.
