# About CheapDesign
CheapDesign (vecchio nome *CheapMC*) è un **pacchetto risorse Minecraft** che cambia le texture in **texture dall’aspetto economico**, facendo apparire il gioco molto “cheap”, come in alcuni *negozi online molto economici*. Il progetto è mantenuto da deutschich (aka. User404_, MineMeeer, MineMeer). Chiunque può contribuire al progetto (maggiori dettagli su [Contributing](contributing.md)).

# Come posso usare questo pacchetto?

Per usare CheapDesign, dipende dalla piattaforma che stai usando.

## Su Windows

Se sei su Windows, puoi usare questo pacchetto sia su **Minecraft: Java Edition** che su **Minecraft: Bedrock Edition** (precedentemente **Minecraft for Windows**).

### Minecraft: Java Edition

Per usare questo pacchetto su **Minecraft: Java Edition**, devi scaricare il pacchetto sui siti di download ufficiali: [Modrinth](https://modrinth.com/project/Jde5u9cx), [CurseForge](https://www.curseforge.com/minecraft/texture-packs/cheapdesign) o [GitHub](https://github.com/deutschich/CheapMC/releases/latest) e posizionare il file ZIP nella cartella `.minecraft\resourcepacks`.

#### Come aprire la cartella `.minecraft`?

Per entrare nella cartella `.minecraft`, devi aprire la **Cerca di Windows** e digitare
```batch
%appdata%\.minecraft
```
. Ora sei nella cartella `.minecraft`.

### Minecraft: Bedrock Edition

Per usare questo pacchetto su **Minecraft: Bedrock Edition**, devi scaricare il pacchetto dal sito di download ufficiale [CurseForge](https://www.curseforge.com/minecraft-bedrock/texture-packs/cheapdesign-bedrock) e aprirlo. Minecraft dovrebbe aprirsi e importare il file.
