# О CheapDesign
CheapDesign (бывшее название *CheapMC*) — это **пак ресурсов Minecraft**, который меняет текстуры на **дешёвые текстуры**, придавая игре очень дешевый вид, как в некоторых *очень дешёвых онлайн‑магазинах*. Проект поддерживается deutschich (также известным как User404_, MineMeeer, MineMeer). Каждый может внести вклад в проект (подробности см. на [Contributing](contributing.md))

# Как использовать этот пакет?

Для использования CheapDesign необходимо учитывать, на какой платформе вы работаете.

## На Windows

Если вы на Windows, вы можете использовать этот пакет в **Minecraft: Java Edition** и/или в **Minecraft: Bedrock Edition** (ранее **Minecraft for Windows**).

### Minecraft: Java Edition

Чтобы использовать этот пакет в **Minecraft: Java Edition**, вам нужно скачать пакет с официальных сайтов скачивания: [Modrinth](https://modrinth.com/project/Jde5u9cx), [CurseForge](https://www.curseforge.com/minecraft/texture-packs/cheapdesign) или [GitHub](https://github.com/deutschich/CheapMC/releases/latest) и поместить ZIP‑файл в папку `.minecraft\resourcepacks`.

#### Как открыть папку `.minecraft`?

Чтобы попасть в папку `.minecraft`, вам нужно открыть **Поиск Windows** и ввести
```batch
%appdata%\.minecraft
```
. Теперь вы в папке `.minecraft`.

### Minecraft: Bedrock Edition

Чтобы использовать этот пакет в **Minecraft: Java Edition**, вам нужно скачать пакет с официального сайта скачивания [CurseForge](https://www.curseforge.com/minecraft-bedrock/texture-packs/cheapdesign-bedrock) и открыть его. Minecraft должен открыть и импортировать файл.

# Могу ли я создать собственные варианты этого пакета?

Да, вы можете создать собственную версию, если добавите явно видимую ссылку на наш репозиторий в качестве атрибуции.
