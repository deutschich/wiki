# О CheapDesign
CheapDesign (старое название *CheapMC*) — это **пак ресурсов Minecraft**, который меняет текстуры на **дорого выглядящие текстуры**, делая игру выглядеть очень дешевой, как на некоторых *очень дешевых онлайн‑магазинах*. Проект поддерживается deutschich (aka. User404_, MineMeeer, MineMeer). Все могут вносить свой вклад в проект (подробнее на [Contributing](contributing.md)).

# Как использовать этот пакет?

Для использования CheapDesign всё зависит от того, на какой платформе вы работаете.

## На Windows

Если вы используете Windows, вы можете использовать этот пакет в **Minecraft: Java Edition** и/или в **Minecraft: Bedrock Edition** (ранее **Minecraft for Windows**).

### Minecraft: Java Edition

Чтобы использовать пакет в **Minecraft: Java Edition**, скачайте его с официальных сайтов загрузки: [Modrinth](https://modrinth.com/project/Jde5u9cx), [CurseForge](https://www.curseforge.com/minecraft/texture-packs/cheapdesign) или [GitHub](https://github.com/deutschich/CheapMC/releases/latest) и поместите ZIP‑файл в папку `.minecraft\resourcepacks`.

#### Как открыть папку `.minecraft`?

Чтобы попасть в папку `.minecraft`, откройте **Поиск Windows** и введите  
```batch
%appdata%\.minecraft
```
. Теперь вы в папке `.minecraft`.

### Minecraft: Bedrock Edition

Чтобы использовать пакет в **Minecraft: Bedrock Edition**, скачайте его с официального сайта загрузки [CurseForge](https://www.curseforge.com/minecraft-bedrock/texture-packs/cheapdesign-bedrock) и откройте. Minecraft должен запустить и импортировать файл.
