## Как внести свой вклад

Спасибо за ваш интерес к вкладу в **CheapDesign**! Мы приветствуем текстуры, созданные сообществом, которые соответствуют дешёвой, низкокачественной эстетике пакета.

### Что вы можете вносить
- Новые или улучшенные текстуры для **Minecraft Java и/или Bedrock Edition**
- Текстуры должны намеренно выглядеть *дешёвыми*, простыми или несовершенными
- Креативные интерпретации приветствуются, если они соответствуют общему стилю

### Руководство по текстурам
- Разрешение: **16×16** (без увеличения масштаба)
- Формат: **PNG** с прозрачностью при необходимости
- Текстуры должны соответствовать формату паков ресурсов **Minecraft Java Edition**
- Разместите все файлы в правильном каталоге внутри `assets/minecraft/textures/` (например, `block/`, `item/`, `entity/` и т.д.)
- Не использовать контент из других наборов текстур, если он не совместим с **CC BY-SA 4.0**
- Не использовать текстуры, созданные ИИ
- Сделайте так, чтобы текстуры были читаемыми в игре

### Лицензия
Этот проект лицензируется под **CC BY-SA 4.0**.  
Внося свой вклад, вы соглашаетесь с тем, что:
- Ваша работа оригинальна или юридически совместима
- Ваш вклад будет опубликован под лицензией **CC BY-SA 4.0**
- Другие могут ремиксировать или распространять вашу работу под той же лицензией

### Как отправлять изменения
1. Форкните официальное репозиторий: https://github.com/deutschich/CheapMC  
2. Создайте новую ветку или используйте свой форк  
3. Добавьте свои текстуры в соответствующие папки **Minecraft Java resource pack**  
4. Создайте **Pull Request в ветку `dev`** ⚠️ Pull Request в другие ветки не будут приняты

### Процесс проверки
Все Pull Request будут рассмотрены на предмет:
- Корректной структуры папок  
- Согласованности стиля  
- Технической корректности  
- Совместимости лицензии
