## Как работает этот пакет для Java и Bedrock  

Я создал этот пакет в основном для Java, но нашёл конвертер Java → Bedrock Pack и использую его. Если хотите добавить текстуры, просто добавьте их в Java Pack через Pull Request. Подробнее об этом можно узнать на [Contributing](contributing).
