## How to contribute

Thank you for your interest in contributing to **CheapDesign**!
We welcome community-made textures that fit the cheap, low-quality aesthetic of the pack.

### What you can contribute
- New or improved textures for **Minecraft Java and/or Bedrock Edition**
- Textures should intentionally look *cheap*, simple, or imperfect
- Creative interpretations are welcome as long as they match the overall style

### Texture guidelines
- Resolution: **16×16** (no upscaling)
- Format: **PNG** with transparency where needed
- Textures must follow the **Minecraft Java Edition resource pack format**
- Place all files in the correct directory inside
  `assets/minecraft/textures/`
  (e.g. `block/`, `item/`, `entity/`, etc.)
- No content from other texture packs unless compatible with **CC BY-SA 4.0**
- Do not use AI-generated textures
- Keep textures readable in-game

### How to submit changes
1. Fork the official repository:
   https://github.com/deutschich/CheapMC
2. Create a new branch or use your fork
3. Add your textures to the correct **Minecraft Java resource pack folders**
4. Create a **Pull Request into the `dev` branch**
   ⚠️ Pull Requests to other branches will not be accepted

### Review process
All Pull Requests will be reviewed for:
- Correct folder structure
- Style consistency
- Technical correctness
- License compatibility

Thank you for helping improve CheapDesign!
